# Bike Sales Dashboard

## Project Overview

Analyzed customer demographics and purchasing behavior to identify factors influencing bike purchases.

## Tools Used

- Power BI
- DAX
- Excel

## KPIs

- Total Customers
- Total Buyers
- Conversion Rate
- Average Income

## Key Insights

- Middle-aged customers purchased most bikes
- Short commute distances had highest conversions
- High-income customers showed stronger purchase rates
