# E-Commerce Sales Analytics Dashboard

## Overview

This project analyzes e-commerce transactions using MySQL and Power BI to identify revenue drivers, profitability trends, and regional performance.

## Tools Used

* MySQL
* Power BI
* DAX

## KPIs

* Total Sales
* Total Profit
* Total Orders
* Profit Margin
* YoY Growth
* MoM Growth
* Running Sales

## Business Insights

* Technology generated the highest revenue.
* West region contributed the largest share of sales.
* Copiers delivered the highest profit.
* Tables generated losses despite strong revenue.

## Skills Demonstrated

* Data Cleaning
* SQL Validation
* Data Modeling
* DAX Measures
* Dashboard Design
* Business Storytelling
